# D40 Image Prompt Plan

This prompt-plan guidance is for future daily image generation and regeneration. Apply it inside the image prompt, not as a replay-page HTML/CSS overlay.

## Summary Cover

- Role: Summary Cover for 里昂 → 日内瓦 → 米兰 → 威尼斯.
- Preferred visible day label: `第40天` if a day label is needed.
- Typography rule: any day label or route typography must be small, understated, secondary, and not dominant.
- Typography must avoid large dominant D36 lettering in generated images.
- Keep Summary Cover text short and sparse: optional day label plus concise route only.
- Use no dense small text, no long Chinese paragraphs, and no typographic clutter.
- Visual priority: route atmosphere, geography, and landmark composition should dominate over text.
- Negative prompt: no billboard-like title treatment, no huge poster headline, no centered giant day code.

## Reuse Rule

- Apply this typography policy to every daily summary cover prompt-plan unless a human explicitly requests strong poster typography.
- Waypoint postcards and special-view images should usually avoid visible day labels entirely.
